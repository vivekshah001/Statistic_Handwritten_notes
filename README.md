# Statistic_Handwritten_notes



Welcome to my Statistics Handwritten Notes collection — a comprehensive, exam-ready and data-analysis-oriented compilation created from scratch with clarity and precision.

These notes are designed to help students, beginners, and data enthusiasts grasp the core concepts of Statistics through structured explanations and handwritten summaries.


| Unit | Topic                                | Description                                                                   |
| ---- | ------------------------------------ | ----------------------------------------------------------------------------- |
| 1    | **Introduction to Statistics**       | Definition, importance, types of data, population vs sample                   |
| 2    | **Measures of Central Tendency**     | Mean, Median, Mode with formulas and examples                                 |
| 3    | **Measures of Dispersion**           | Range, Variance, Standard Deviation, Coefficient of Variation                 |
| 4    | **Correlation & Regression**         | Karl Pearson’s coefficient, Spearman’s rank, regression lines                 |
| 5    | **Probability Basics**               | Events, sample space, conditional probability, Bayes theorem                  |
| 6    | **Random Variables & Distributions** | Discrete & continuous distributions, Binomial, Poisson, Normal                |
| 7    | **Sampling & Estimation**            | Sampling techniques, sampling errors, point and interval estimation           |
| 8    | **Hypothesis Testing**               | Null & alternative hypothesis, Type I & II errors, Z-test, t-test, chi-square |
| 9    | **Applications in Data Analysis**    | Using statistics in data analytics, business, and ML context                  |
